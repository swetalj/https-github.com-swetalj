<h2><a href="https://www.geeksforgeeks.org/problems/binary-tree-to-dll/1?page=4&company=Morgan+Stanley&sortBy=difficulty">Binary Tree to DLL</a></h2><h3>Difficulty Level : Difficulty: Hard</h3><hr><div class="problems_problem_content__Xm_eO"><p data-pm-slice="0 0 []">G<span style="font-size: 14pt;">iven a <strong>root</strong> of binary tree (BT), convert it to a Doubly Linked List (DLL) in place using the <strong>same node</strong> structure. The<strong> left </strong>and <strong>right</strong> pointers in the binary tree nodes should be used as <strong>prev</strong> and <strong>next</strong> pointers respectively in the resulting DLL .The DLL should be formed by performing an <strong>inorder</strong> traversal of the binary tree (i.e., Left → Root → Right).The <strong>first node </strong>in the inorder traversal (i.e., the leftmost node) should become the <strong>head</strong> of the DLL. Return the <strong>head</strong> of the resulting DLL.<br></span><span style="font-size: 14pt;"><strong>Note: </strong>h&nbsp;is the tree's height, and this space is used implicitly for the recursion stack.</span></p>
